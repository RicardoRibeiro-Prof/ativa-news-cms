# Ativa News CMS

Portal de notícias regional com painel administrativo e integração ao Supabase.

## Stack

- HTML5
- CSS3
- JavaScript ES Modules
- Supabase Auth
- Supabase Database
- Supabase Storage

## Supabase

Projeto configurado para usar:

- URL: `https://gqpasfkqbgjhqcfsxuwv.supabase.co`
- Bucket público: `portal-images`

## Estrutura inicial

- `index.html`: portal público
- `admin/login.html`: autenticação administrativa
- `admin/dashboard.html`: painel protegido
- `assets/css/`: estilos
- `assets/js/`: scripts do portal e painel
- `supabase/`: cliente e serviços do Supabase
